## Список доработок

- [ ] **Кнопка «Профиль» → имя пользователя** — в шапке (`components/Header.tsx`) вместо текста «Профиль» показывать имя пользователя (или «Имя», если имя не задано). Данные брать из сессии / `/api/profile`.

- [ ] **Поиск городов** — плохо ищет города, например «Люберцы» не находит. Нужно улучшить поиск/автокомплит городов (`lib/utils/places.ts`, `CityPicker`).

- [ ] **Сохранение данных пользователя** — если пользователь один раз заполнил свои данные (имя, дата рождения, город и т.д.), сохранять их в MongoDB и при следующем визите показывать/подставлять пользователю (модель `User` / `Client`, `/api/profile`).
